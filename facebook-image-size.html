Facebook Sizes That Matter

For Facebook feed posts, 1200 x 630 px is the standard wide format. Stories use 1080 x 1920 px, while cover photos need a very wide crop that can display differently across devices.

Resize before uploading when the crop matters, especially for covers, ads and link-preview style posts.

Quick Export Workflow

Pick the Facebook placement first. A feed image, Story and cover photo use very different shapes, so resizing one image for all placements usually creates poor crops. Export a separate version for each placement when the composition matters.

For covers and feed graphics with text, keep the important message away from the edges. Facebook may show previews differently on mobile and desktop, and centered content survives those changes better. After resizing, compress the final JPG if upload speed or file size is a concern.

Frequently Asked Questions

What is the best Facebook image size?
Use 1200 x 630 px for standard feed images, 1080 x 1920 px for Stories and 851 x 315 px for cover photos.
Why does Facebook crop my image?
Facebook displays images in different containers across feed, mobile and desktop. Resizing to the target format gives you better control.
What size should a Facebook cover photo be?
Use 851 x 315 px as a practical desktop cover size, and keep important content near the center because mobile display can crop the sides.
Can I reduce the Facebook image file size?
Yes. Resize first, then use Compress Image to 100KB or the general compressor.
